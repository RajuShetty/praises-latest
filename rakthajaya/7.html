121 ಅಡವಿಯ ಹುಲ್ಲುಗಳಿಗೆ ಉಡಿಸುವರಾಗಿರುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 6:30
122ನಮಗೆ ಬೇಕಾಗಿರುವುದನ್ನು ತಿಳಿದಿರುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 6:32
123ಬಂಡೆಯ ಮೇಲೆ ನಿಲ್ಲಿಸುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 7:24
124ಪಾಪಗಳನ್ನು ಕ್ಷಮಿಸಿದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 9:2
125ಜನರಿಗಾಗಿ ಪರಿತಪಿಸುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮಾರ್ಕ 8:2
126ಹೃದಯದ ಯೋಚನೆಗಳನ್ನು ತಿಳಿದಿರುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 9:4
127ಎಚ್ಚರಿಕೆಯಾಗಿರಿ ಎಂದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 10:17
128ಶಪಿಸುವವರನ್ನು ಆಶೀರ್ವದಿಸಿರಿ ಎಂದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯರೋಮಾ 12:11
129ಪ್ರಾರ್ಥನೆಮಾಡುತ್ತಾ ಎಚ್ಚರಂದಿಂದಿರಿ ಎಂದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಲೂಕ 21:36
130ಶೋಧನೆಗೊಳಗಾಗದಂತೆ ಪ್ರಾರ್ಥನೆ ಮಾಡಿರಿ ಎಂದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯಲೂಕ 22:40
131ಕಾದುಕೊಂಡಿರಿ ಎಂದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಅಪೋ 1:5
132ಆತ್ಮದಲ್ಲಿ ಬೆಂಕಿಯಾಗಿರಿ ಎಂದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ರೋಮಾ 12:11
133ಬಲವುಳ್ಳವರಾಗಿರುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 10:28
134ಶಾಂತತೆಯುಳ್ಳವರಾಗಿರುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ 2ಕೊರಿ 10:1
135ದಯೆಯುಳ್ಳವರಾಗಿರುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ 2ಕೊರಿ 10:1
136ಮನ ಮರುಗಿದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 20:34
137ಪ್ರವಾದಿಯಾಗಿರುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 21:11
138ಆಶೀರ್ವಾದ ಎಂದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 23:39
139ಬಲವಾಗಿ ಬರುವಂತ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 24:30
140ಅಧಿಕವಾದ ಮಹಿಮೆಯಿಂದ ಬರುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯಮತ್ತಾ 24:30
