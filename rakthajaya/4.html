61 ಮಹತ್ಕಾರ್ಯಗಳನ್ನು ಮಾಡುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮಾರ್ಕ 6:2
62ಎಪ್ಪಥಾ ತೆರೆಯಲ್ಪಡಲಿ ಎಂದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮಾರ್ಕ 7:34
63ಕಳವಳಗೊಳ್ಳದಿರಿ ಎಂದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 6:34
64ಕೇಳಿರಿ, ಕೊಡಲ್ಪಡುವದು ಎಂದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 7:7
65ಹುಡುಕಿರಿ, ಸಿಗುವುದು ಎಂದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 7:7
66ತಟ್ಟಿರಿ ತೆರೆಯಲಾಗುವುದು ಎಂದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 7:7
67ಕೇಳುವ ಪ್ರತಿಯೊಬ್ಬನು ಹೊಂದಿಕೊಳ್ಳುವನು ಎಂದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 7:8
68ಹುಡುಕುವವನು ಕಂಡುಕೊಳ್ಳುತ್ತಾನೆ ಎಂದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 7:8
69ತಟ್ಟುವವನಿಗೆ ತರೆಯಲ್ಪಡುವುದು ಎಂದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 7:8
70ಸಮುದ್ರದ ದಡದಲ್ಲಿ ಉಪದೇಶ ಮಾಡಿದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮಾರ್ಕ 4:1
71ಅಧಿಕಾರವುಳ್ಳವರಾಗಿ ಭೋಧಿಸುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 7:28
72ಸಾಮ್ಯರೂಪವಾಗಿ ಬೋಧಿಸುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮಾರ್ಕ 4:2
73ವಾಕ್ಯವನ್ನು ಬೋಧಿಸುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮಾರ್ಕ 2:2
74ಸಬ್ಬತ್ ದಿನಕ್ಕೆ ದೇವರಾಗಿರುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮಾರ್ಕ 2:28
75ಸಬ್ಬತ್ ದಿನದಲ್ಲಿ ಪ್ರಸಂಗ ಮಾಡಿದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮಾರ್ಕ 1:21
76ಸ್ವಸ್ಥಮಾಡಿದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮಾರ್ಕ 3:2
77ಪರಿಶುದ್ದಾತ್ಮನಿಂದ ಅಭಿಷೇಕ ಮಾಡುವವರಾಗಿರುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 3:11
78ಅಗ್ನಿಯಿಂದ ಅಭಿಷೇಕ ಮಾಡುವವರಾಗಿರುವ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 3:11
79ರಾಜ್ಯದ ಸುವಾರ್ತೆಯನ್ನು ಪ್ರಸಂಗ ಮಾಡಿದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯ ಮತ್ತಾ 4:23
80ಸ್ವಸ್ಥಮಾಡುವೆನು ಎಂದ ಯೇಸು ಕ್ರಿಸ್ತನೇ ನಿಮ್ಮ ರಕ್ತಕ್ಕೆ ಜಯಮತ್ತಾ 8:7
